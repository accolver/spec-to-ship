# External Impeccable Install

Canonical source: https://github.com/pbakaus/impeccable

Preferred install command:

```bash
npx impeccable skills install
```

This is an external network/install action. Ask before running it.

After install, run `/impeccable init` inside the AI coding tool.

If install is blocked, pause for user decision or proceed only with a clearly labeled degraded UI/UX gate and documented risk.
