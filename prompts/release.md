# STS Release Prompt

Read `SPEC-TO-SHIP.md`, load the relevant STS skill, and work inside the active `.spec-to-ship/features/<feature-id>/` namespace.

Release readiness is blocked by unresolved review blockers, failed or unavailable required validation, missing UI/demo evidence, unreviewed dependency changes, dirty unrelated artifacts, or unapproved degraded mode. Record each validation as passed, failed, or not run/unavailable before claiming readiness.
