# Onboarding Screen Spec

## Goal
Create a clear first-run onboarding screen that helps new users understand the product value, complete account setup, and continue into the app.

## Proposed experience
- Hero heading: "Welcome — let's get you set up"
- Short explanation of what the app will help the user do
- Three setup steps with completion indicators:
  1. Confirm profile details
  2. Connect required account data
  3. Choose notification preferences
- Primary CTA: "Start setup"
- Secondary CTA: "Skip for now"

## UX requirements
- Mobile and desktop responsive layout
- Visible loading state while account data loads
- Empty state if setup steps cannot be fetched
- Error state with retry if onboarding status fails
- Accessible labels for buttons and progress indicators

## Missing design input
Impeccable is not installed in this environment, so I would proceed using standard product and accessibility heuristics and mark the design for later review once that tool is available.
